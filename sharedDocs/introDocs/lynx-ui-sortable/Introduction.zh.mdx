## 结构

整个 `SortableItem` 都可以被拖拽。

```jsx
 <SortableRoot data={data}>
    {(item: SortableData<string>) => {
      return (
        <SortableItem />
      )
    }}
 </SortableRoot>
```

只有 `SortableItemArea` 可以被拖拽。

```jsx
 <SortableRoot data={data}>
    {(item: SortableData<string>) => {
      return (
        <SortableItem as='DraggableRoot' >
          <SortableItemArea />
        </SortableItem>
      )
    }}
 </SortableRoot>
```
