# `<FeedList>`

一个基于[`<List>`](/lynx-ui/Components/List.html)封装的可滚动容器。
增加基于 MTS 实现的刷新和回弹能力，同时支持自定义加载更多状态的底部内容，以及数据加载完毕状态的底部内容。

:::info

你可以查看[源码](https://github.com/lynx-family/lynx-ui/tree/main/packages/lynx-ui-feed-list)来了解它的运行原理，也欢迎大家提 MR 来丰富其能力。

此外，你也可以参考这个实现，利用 [MTS](/zh/lynx-ui/Guides/MainThreadScript.html) 自行实现定制化的列表刷新 / 加载更多能力。

:::
