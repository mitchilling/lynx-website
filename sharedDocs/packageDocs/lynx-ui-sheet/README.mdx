# `<Sheet>`

{lynxUiIntros['lynx-ui-sheet']}

- Supports controlled and uncontrolled visibility.
- Supports snap points, including percentages and content-driven height.
- Supports imperative control through `SheetRootRef`.

:::info

You can check the [source code](https://github.com/lynx-family/lynx-ui/tree/main/packages/lynx-ui-sheet) to understand how it works, and you're welcome to submit MRs to enhance its capabilities.

:::
