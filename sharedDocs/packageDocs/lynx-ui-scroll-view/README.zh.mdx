# `<Scrollview>`

{lynxUiIntrosZh['lynx-ui-scroll-view']}

- 利用 [`LazyComponent`](/lynx-ui/components/lazy-component.html)内置懒加载能力的滚动容器，用于优化首屏耗时；
- 内置双端对齐的 bounces 能力。

:::info

你可以查看[源码](https://github.com/lynx-family/lynx-ui/tree/main/packages/lynx-ui-scroll-view)来了解它的运行原理，也欢迎大家提 MR 来丰富其能力。

:::
