# markState

<APISummary />

Mark the current TemplateData with the associated dataProcessor name. When this TemplateData is used in [UpdateData](/api/lynx-native-api/lynx-view/update-meta-data), the corresponding dataProcessor will be found based on this name for data preprocessing.

## Syntax

### Android

```java
public void markState(String name);
```

#### Parameters

- `name`: Marked name.

### iOS

```objc
- (void)markState:(NSString *)name;
```

#### Parameters

- `name`: Marked name.
