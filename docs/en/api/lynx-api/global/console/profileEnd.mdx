# console: profileEnd() method

<APISummary />

`profileEnd` stops recording a profile previously started with console.profile().

## Syntax

```ts
profileEnd(profileEndName: string): undefined;
```

### Parameters

#### profileEndName

The name to give the profile.

### return value

None(`undefined`).
