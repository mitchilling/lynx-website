# console: timeEnd() method

<APISummary />

`timeEnd` stops a timer that was previously started by calling console.time().

## Syntax

```ts
timeEnd(label: string?): undefined;
```

### Parameters

#### label

A string representing the name to give the new timer.

### return value

None(`undefined`).
