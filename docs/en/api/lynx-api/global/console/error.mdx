# console: error() method

<APISummary />

`error` outputs a message to the console at the "error" log level.

## Syntax

```ts
error(val1, val2, ...): undefined;
```

### Parameters

#### val1, val2, ...

A list of JavaScript values to output.

### return value

None(`undefined`).
