# lynx: cancelAnimationFrame() static method

<APISummary />

Cancel a [`requestAnimationFrame`] callback.

## Syntax

```ts
cancelAnimationFrame: (requestID: number) => void;
```

### Parameters

#### requestID

The return value of the [`requestAnimationFrame`] that needs to be cancelled.

### Return Value

None (`undefined`).
