## Syntax

```css
font-style: italic;
font-style: oblique;
font-style: normal;
```

### Values

- **`normal`**
  Selects a font that is classified as normal within a font-family.

- `italic`
  Selects a font that is classified as `italic`.

- `oblique`
  Selects a font that is classified as `oblique`.


## Difference with Web

- Only support `normal`, `oblique` and `italic`
