## Introduction

Used to set the centering alignment of the current element in the [relative-layout](/guide/ui/layout/relative-layout.mdx).

## Value

### `none`

**Default Value**. Not centered.

### `both`

Centered vertically and horizontally.

### `vertical`

Center vertically.

### `horizontal`

Centered horizontally.

## Syntax

```css
relative-center: none;
relative-center: both;
relative-center: vertical;
relative-center: horizontal;
```
