**App frameworks** built on Lynx offer a complete development experience for shipping a new native application end to end, with native navigation, lifecycle management, and production-ready tooling, ideal for **Greenfield** projects.

### Sparkling <PlatformBadge platform="ios" /> <PlatformBadge platform="android" />

**Lynx at TikTok scale.**
[Sparkling](https://tiktok.github.io/sparkling/) is TikTok's open-source mobile application framework built on Lynx, bringing the same cross-platform infrastructure that powers TikTok itself to developers. It pairs a type-safe JavaScript-to-native bridge with scheme-driven multi-page routing and project scaffolding, deploys to both **iOS and Android** from a single bundle, and is production-tested across TikTok's ecosystem. Visit the [Sparkling documentation](https://tiktok.github.io/sparkling/) to get started.

### Lynxtron <Badge text="Coming soon" type="info" />

**Lynx for desktop.**
Lynxtron is the upcoming desktop counterpart, an Electron-like framework with a lightweight Lynx renderer that ships the same UI across **macOS, Windows, and Web**, extensible through Node-API native modules.
