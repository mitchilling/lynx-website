# Lynx 配置

## 可配置项

[`@lynx-js/type-config`](https://www.npmjs.com/package/@lynx-js/type-config) 定义了 Lynx 的全部配置项。每个配置项都提供了完善的类型注释，用于说明其行为与默认值。同时我们也提供了更直观的[配置项参考](./config-reference.mdx)。

## @lynx-js/config-rsbuild-plugin

在使用具体前端框架时，`pluginReactLynx` 或其他 DSL 插件通常只暴露部分配置能力。若需要使用 Lynx 的完整配置项，请使用 [`@lynx-js/config-rsbuild-plugin`](https://www.npmjs.com/package/@lynx-js/config-rsbuild-plugin)。

```ts title="lynx.config.ts"
import { pluginLynxConfig } from '@lynx-js/config-rsbuild-plugin';
import { defineConfig } from '@lynx-js/rspeedy';

export default defineConfig({
  plugins: [
    pluginLynxConfig({
      enableCheckExposureOptimize: false,
    }),
  ],
});
```
