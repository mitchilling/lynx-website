## 属性

### `bounces`

 <IOSOnly />

```tsx
// @defaultValue: false
bounces?: boolean;
```

启用弹性效果


### `cookies`

 <ClayOnly /> <VersionBadge>Lynx 3.5</VersionBadge>

```tsx
cookies?: any;
```

预设 cookies


### `enable-debug`

 <AndroidOnly /> <IOSOnly /> <ClayOnly /> <HarmonyOnly />

```tsx
// @defaultValue: false
'enable-debug'?: boolean;
```

启用 WebView 调试，可在 Chrome DevTools 中进行调试


### `html`

 <AndroidOnly /> <IOSOnly /> <HarmonyOnly /> <VersionBadge>3.6</VersionBadge>

```tsx
html?: string;
```

表示要加载的 HTML 内容的字符串。当 html 变化时会自动触发内容刷新。优先级低于 `src`。


### `initjs`

 <ClayOnly /> <VersionBadge>Lynx 3.5</VersionBadge>

```tsx
initjs?: string;
```

在文档就绪时执行的 JavaScript


### `params`

 <AndroidOnly /> <IOSOnly />

```tsx
params?: object;
```

外部 webview 实现的参数


### `scroll-bar-enable`

 <IOSOnly />

```tsx
// @defaultValue: false
'scroll-bar-enable'?: boolean;
```

启用滚动条


### `src`

 <AndroidOnly /> <IOSOnly /> <ClayOnly /> <HarmonyOnly />

```tsx
src?: string;
```

表示远程服务器上资源位置的字符串。当 src 变化时会自动触发内容刷新。


### `use-osr`

 <ClayOnly /> <VersionBadge>Lynx 3.5</VersionBadge>

```tsx
// @defaultValue: false
'use-osr'?: boolean;
```

是否启用离屏渲染模式


### `webview-type`

 <AndroidOnly /> <IOSOnly /> <HarmonyOnly />

```tsx
// @defaultValue: 'default'
'webview-type'?: string;
```

指定 webview 的类型，可以是通过 LynxService 注入的 webview 实现

## 方法

前端可以通过 [SelectorQuery](/api/lynx-api/nodes-ref/nodes-ref-invoke.html) API 调用组件方法。

### `cookies.flushStore`

 <ClayOnly /> <VersionBadge>Lynx 3.5</VersionBadge>
```ts

lynx.createSelectorQuery()
     .select('#id')
     .invoke({
      method: 'cookies.flushStore',
      success: function (res) {},
      fail: function (res) {},
    })
    .exec();

```

将所有未写入的 cookies 数据刷新到磁盘


### `cookies.get`

 <ClayOnly /> <VersionBadge>Lynx 3.5</VersionBadge>
```ts

lynx.createSelectorQuery()
     .select('#id')
     .invoke({
      method: 'cookies.get',
      params: {
        /**
         * 获取域名匹配或为其子域名的 cookies
         * @PC
         */
        domain?: string;
        /**
         * 按 httpOnly 过滤 cookies
         * @PC
         */
        httpOnly?: boolean;
        /**
         * 按名称过滤 cookies
         * @PC
         */
        name?: string;
        /**
         * 获取路径匹配的 cookies
         * @PC
         */
        path?: string;
        /**
         * 按 Secure 属性过滤 cookies
         * @PC
         */
        secure?: boolean;
        /**
         * 过滤会话 cookie 或持久化 cookie
         * @PC
         */
        session?: boolean;
        /**
         * 获取与指定 URL 关联的 cookies。为空则获取所有 URL 的 cookies
         * @PC
         */
        url?: string;
      };
      success: function (res) {},
      fail: function (res) {},
    })
    .exec();

```

从 webview 获取 cookies

```ts
lynx.createSelectorQuery()

```

### `cookies.set`

 <ClayOnly /> <VersionBadge>Lynx 3.5</VersionBadge>
```ts

lynx.createSelectorQuery()
     .select('#id')
     .invoke({
      method: 'cookies.set',
      params: {
        /**
         * cookie 的域名。省略时默认为空
         * @PC
         */
        domain?: string;
        /**
         * cookie 的过期时间，为 UNIX 纪元以来的秒数
         * @PC
         */
        expirationDate?: number;
        /**
         * cookie 是否标记为 HTTP only
         * @PC
         * @defaultValue false
         */
        httpOnly?: boolean;
        /**
         * cookie 的名称
         * @PC
         */
        name: string;
        /**
         * cookie 的路径。省略时默认为空
         * @PC
         */
        path?: string;
        /**
         * 应用于此 cookie 的 Same Site 策略。可为 unspecified、no_restriction、lax 或 strict
         * @PC
         * @defaultValue 'lax'
         */
        sameSite?: string;
        /**
         * cookie 是否标记为 secure
         * @PC
         * @defaultValue false
         */
        secure?: boolean;
        /**
         * 关联 cookie 的 URL
         * @PC
         */
        url: string;
        /**
         * cookie 的值。省略时默认为空
         * @PC
         */
        value?: string;
      };
      success: function (res) {},
      fail: function (res) {},
    })
    .exec();

```

为 webview 设置 cookie
