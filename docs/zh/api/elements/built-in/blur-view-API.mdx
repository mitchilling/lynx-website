## 属性

### `android-capture-target`

 <AndroidOnly /> <VersionBadge>3.9</VersionBadge>

```tsx
'android-capture-target'?: string;
```

用于指定作为模糊源的 Android Lynx view 的 raw id，仅取其与 `<blur-view>` 相交的区域作为模糊内容。当未设置或无法解析时，Android 使用默认的父节点捕获路径。


### `blur-effect`

 <IOSOnly />

```tsx
// @默认值: 'light'
'blur-effect'?: 'light' | 'extra-light' | 'dark' | 'glass' | 'glass-container';
```

主要影响模糊区域的明度：
light：明度与背景基本一致；
dark：明度比背景更暗；
extra-light：明度比背景更亮；
glass：渲染为玻璃材质的视觉效果；
glass-container：使用 UIGlassContainerEffect 将多个 glass 元素合成为一个组合效果。


### `blur-radius`

 <AndroidOnly /> <IOSOnly /> <HarmonyOnly />

```tsx
// @默认值: "0px"
'blur-radius'?: string;
```

高斯模糊半径。


### `blur-sampling`

 <AndroidOnly />

```tsx
// @默认值: 6
'blur-sampling'?: number;
```

模糊区域的降采样比例，相当于先对背景区域进行缩小模糊再放大还原。会对模糊效果有轻微影响，但能显著提升性能。


### `enable-auto-blur`

 <AndroidOnly />

```tsx
// @默认值: true
'enable-auto-blur'?: boolean;
```

是否自动更新模糊效果。


### `experimental-update-blur-radius`

 <AndroidOnly /> <VersionBadge>3.4</VersionBadge>

```tsx
'experimental-update-blur-radius'?: boolean;
```

切换 Android 内部模糊 buffer 的刷新路径。


### `glass-interactive`

 <IOSOnly /> <VersionBadge>3.8</VersionBadge>

```tsx
// @默认值: false
'glass-interactive'?: boolean;
```

启用玻璃效果的交互行为。


### `glass-style`

 <IOSOnly /> <VersionBadge>3.8</VersionBadge>

```tsx
// @默认值: 'regular'
'glass-style'?: 'clear' | 'regular';
```

玻璃效果的样式。


### `glass-tint-color`

 <IOSOnly /> <VersionBadge>3.8</VersionBadge>

```tsx
// @默认值: 'transparent'
'glass-tint-color'?: string;
```

施加在玻璃效果上的着色（tint color）。


### `spacing`

 <IOSOnly />

```tsx
// @默认值: 0
spacing?: number;
```

指定元素之间开始融合的间距阈值。
