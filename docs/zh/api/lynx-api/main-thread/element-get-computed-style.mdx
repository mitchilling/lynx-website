# getComputedStyleProperty()

<APISummary />

## 介绍

[`Element`](./main-thread-element.mdx) 接口的 **`getComputedStyleProperty()`** 方法返回元素的指定[计算样式](https://developer.mozilla.org/zh-CN/docs/Web/CSS/Guides/Cascade/Property_value_processing#resolved_value)。

## 语法

```ts
const value = element.getComputedStyleProperty(styleName);
```

### 参数

#### styleName

指定要获取的属性名的字符串。样式名必须是 [kebab-case](https://developer.mozilla.org/zh-CN/docs/Glossary/Kebab_case)。

### 返回值

包含[计算样式](https://developer.mozilla.org/zh-CN/docs/Web/CSS/Guides/Cascade/Property_value_processing#resolved_value)值的字符串。

## 示例

```ts
const width = element.getComputedStyleProperty('width');
const backgroundColor = element.getComputedStyleProperty('background-color');
```
