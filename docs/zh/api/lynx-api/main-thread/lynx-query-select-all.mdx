# lynx: querySelectorAll(selector: string)

<APISummary />

在页面的子元件中查找所有满足指定选择器的元件。

## 语法

```ts
const elementArray = lynx.querySelectorAll(selector);
```

### 参数

#### selector

指定选择器，支持的选择器列表见 [selector](../selector-query/selector-query-select#selector)。

### 返回值

[`MainThread.Element[]`](../main-thread/main-thread-element) 或 `[]`。
