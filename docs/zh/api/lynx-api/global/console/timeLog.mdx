# console: timeLog() method

<APISummary />

`timeLog` 方法用于记录以前通过调用 console.Time() 启动的计时器的当前值。

## 语法

```ts
timeLog(label: string?): undefined;
```

### 参数

#### label

timer 的名称。

### 返回值

无（`undefined`）。
