# console: time() method

<APISummary />

`time` 方法用于启动一个计时器，您可以用来跟踪操作需要多长时间。

## 语法

```ts
time(label: string?): undefined;
```

### 参数

#### label

timer 的名称。

### 返回值

无（`undefined`）。
