# console: debug() method

<APISummary />

`debug` 方法用于在控制台面板输出信息，日志级别为 debug 级别。

## 语法

```ts
debug(val1, val2, ...): undefined;
```

### 参数

#### val1, val2, ...

输出的 JavaScript 值列表。这些值中的每个值的表示形式都以给定的顺序输出。

### 返回值

无（`undefined`）。
