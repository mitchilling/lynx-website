基于 Lynx 构建的 **应用框架（App Framework）** 提供端到端的开发体验，帮助你从零交付一个完整的原生应用，内置原生导航、生命周期管理和生产级工具链，适合**全新（Greenfield）**项目。

### Sparkling <PlatformBadge platform="ios" /> <PlatformBadge platform="android" />

**TikTok 级 Lynx 设施。**
[Sparkling](https://tiktok.github.io/sparkling/) 是 TikTok 基于 Lynx 开源的移动端框架，将驱动 TikTok 的跨平台基础设施带给开发者。它提供类型安全的 JavaScript 到原生桥接、基于 scheme 的多页面路由以及项目脚手架，从同一份代码包同时部署到 **iOS 和 Android**，并已在 TikTok 生态中经过大规模生产验证。访问 [Sparkling 文档](https://tiktok.github.io/sparkling/) 开始构建移动应用。

### Lynxtron <Badge text="即将推出" type="info" />

**桌面端的 Lynx 设施。**
Lynxtron 是即将推出的桌面端框架，一个轻量的 Electron 替代方案，以 Lynx 渲染同一份 UI 跨越 **macOS、Windows 和 Web**，并支持通过 Node-API 扩展原生模块。

:::tip 开发提示
在开发过程中，你可以使用 **Lynx Explorer** 快速预览和迭代 Lynx 页面。切换到[用 Explorer 体验](/zh/guide/start/quick-start)进行设置。
:::
