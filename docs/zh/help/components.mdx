# MDX 组件参考

本页作为 Lynx 文档中可用自定义组件的参考。您可以从 `@lynx` 导入这些组件。

## 标注与警告 (Callout & Alerts)

用于突出显示重要信息。您可以使用标准 Markdown 语法 `:::type` 来创建标注。

```markdown
:::info
这是一个信息标注。
:::

:::warning
这是一个警告标注。
:::

:::tip
这是一个提示标注。
:::

:::danger
这是一个危险标注。
:::
```

:::info
这是一个信息标注。
:::

:::warning
这是一个警告标注。
:::

:::tip
这是一个提示标注。
:::

:::danger
这是一个危险标注。
:::

## 徽章 (Badges)

### 平台徽章 (Platform Badges)

指示平台支持情况。

```tsx
<AndroidOnly />
<IOSOnly />
<WebOnly />
<HarmonyOnly />
```

<div style={{ display: 'flex', gap: '8px', flexWrap: 'wrap' }}>
  <AndroidOnly />
  <IOSOnly />
  <WebOnly />
  <HarmonyOnly />
</div>

### 状态徽章 (Status Badges)

指示 API 或功能状态。

```tsx
<Deprecated />
<Experimental />
<Required />
```

<div style={{ display: 'flex', gap: '8px', flexWrap: 'wrap' }}>
  <Deprecated />
  <Experimental />
  <Required />
</div>

### 运行时徽章 (Runtime Badges)

指示脚本在哪个线程上运行。

```tsx
<RuntimeBadge type="mts" />
<RuntimeBadge type="bts" />
```

<div style={{ display: 'flex', gap: '8px', flexWrap: 'wrap' }}>
  <RuntimeBadge type="mts" />
  <RuntimeBadge type="bts" />
</div>

### 版本徽章 (Version Badge)

指示引入某个功能的 Lynx 版本。

```tsx
<VersionBadge v={2.5} />
<VersionBadge>2.5.1</VersionBadge>
```

<div style={{ display: 'flex', gap: '8px', flexWrap: 'wrap' }}>
  <VersionBadge v={2.5} />
  <VersionBadge>2.5.1</VersionBadge>
</div>

### 通用徽章 (Generic Badges)

标准 Rspress 徽章。

```tsx
<Badge type="tip" text="Recommended" />
<Badge type="warning" text="Deprecated" />
```

<div style={{ display: 'flex', gap: '8px' }}>
  <Badge type="tip" text="Recommended" />
  <Badge type="warning" text="Deprecated" />
</div>

## 布局与容器 (Layout & Containers)

### 平台选项卡 (Platform Tabs)

按平台组织内容。

```tsx
<PlatformTabs defaultPlatform="ios" queryKey="platform">
  <PlatformTabs.Tab platform="ios">iOS 内容</PlatformTabs.Tab>
  <PlatformTabs.Tab platform="android">Android 内容</PlatformTabs.Tab>
</PlatformTabs>
```

<PlatformTabs defaultPlatform="ios" queryKey="platform">
  <PlatformTabs.Tab platform="ios">iOS 内容</PlatformTabs.Tab>
  <PlatformTabs.Tab platform="android">Android 内容</PlatformTabs.Tab>
</PlatformTabs>

### 列 (Columns)

将内容分为多列。传递 `titles` 属性以设置列标题。

```tsx
<Columns titles={['左列', '右列']}>
  <div>左列内容。</div>
  <div>右列内容。</div>
</Columns>
```

<Columns titles={['左列', '右列']}>
  <div>左列内容。</div>
  <div>右列内容。</div>
</Columns>

### 响应式双列 (Responsive Dual Column)

一种在小屏幕上切换为单列的布局。使用 `FlexItem` 控制宽度。

```tsx
<ResponsiveDualColumn>
  <FlexItem minWidth={300}>
    <div>左侧内容 (min 300px)</div>
  </FlexItem>
  <FlexItem minWidth={300}>
    <div>右侧内容 (min 300px)</div>
  </FlexItem>
</ResponsiveDualColumn>
```

<ResponsiveDualColumn>
  <FlexItem minWidth={300}>
    <div style={{ background: 'var(--rp-c-bg-mute)', padding: '1rem' }}>
      左侧内容 (min 300px)
    </div>
  </FlexItem>
  <FlexItem minWidth={300}>
    <div style={{ background: 'var(--rp-c-bg-mute)', padding: '1rem' }}>
      右侧内容 (min 300px)
    </div>
  </FlexItem>
</ResponsiveDualColumn>

### 浏览器容器 (Browser Container)

将内容包装在类浏览器窗口框架中。

```tsx
<BrowserContainer>
  <div style={{ padding: '20px', textAlign: 'center' }}>浏览器内容</div>
</BrowserContainer>
```

<BrowserContainer>
  <div style={{ padding: '20px', textAlign: 'center' }}>浏览器内容</div>
</BrowserContainer>

### 代码折叠 (CodeFold)

可折叠的代码块，可选择包含图像。

````tsx
<CodeFold toggle>```ts console.log('隐藏代码'); ```</CodeFold>
````

<CodeFold toggle>

```ts
console.log('隐藏代码');
```

</CodeFold>
